# BroadStock (jaego) 처음 설정

## 1) Supabase 프로젝트

1. [supabase.com](https://supabase.com) 에서 프로젝트 생성  
2. **Settings → API** 에서 `Project URL`, `anon public` 키 복사  
3. **Authentication → Users** → **Create user**  
   - Email: `broadstock-kiosk@example.com` (또는 본인 이메일 — 그 경우 아래 env에 동일하게)  
   - Password: `159311` (코드의 `KIOSK_PIN` 과 동일)

## 2) SQL 실행

**SQL Editor** 에서 `supabase/migrations/001_initial.sql` 전체를 붙여 넣고 **Run**.

## 3) 로컬 환경 변수

`.env.local.example` 을 복사해 `.env.local` 로 저장하고 URL/키를 넣습니다.

```powershell
cd c:\Users\COM\jaego
copy .env.local.example .env.local
npm run setup:check
npm run dev
```

## 4) Vercel

1. GitHub에 `jaego` 저장소로 푸시  
2. Vercel → Import → 같은 env 변수를 **Production** 에 넣고 배포  
3. **Authentication → URL Configuration** 에 배포 URL을 **Site URL** / **Redirect URLs** 에 추가
